Controlador wifi de luces led

Para este trabajo nos busco un cliente que requeria actualizar su controlador de luces T-8000 TTL el cual requiere de una programacion anticipada de las secuencias que se quieren ver en las luces led, para solucionar el requerimiento de nuestro cliente se programaran dos nodos usando el esp32 el cual puede controlar hasta casi los 6400 leds usando el programa wled.
